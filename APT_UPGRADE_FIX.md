# РЕШЕНО: Ошибка "apt-get upgrade отключена" на Astra Linux 1.7

## Проблема

При запуске:
```bash
sudo ./scripts/deploy.sh all .deployment.conf
```

Вы видите ошибку:
```
E: 'apt-get upgrade' отключена, поскольку это может привести систему в нерабочее состояние.
Вместо этого рекомендуется использовать 'apt-get upgrade', несмотря на это предупреждение,
используйте опцию '--enable-upgrade' или включите настройку конфигурации APT::Get::EnableUpgrade
```

## ✅ Решение

### Это НОРМАЛЬНО!

На Astra Linux 1.7 с уровнем защиты **"Орёл"** полное обновление пакетов отключено по умолчанию в целях безопасности. **Скрипты развёртывания уже обновлены для обработки этой ошибки.**

### Что делать?

**Ничего!** Просто дождитесь завершения скрипта развёртывания. Ошибка будет игнорирована и процесс продолжится.

### Если вы хотите вручную исправить

Используйте вспомогательный скрипт:

```bash
# Сделать исполняемым
chmod +x fix_apt_upgrade.sh

# Запустить
sudo ./fix_apt_upgrade.sh
```

Он предложит несколько вариантов решения:

1. **Обновить пакеты один раз** - использует опцию APT::Get::EnableUpgrade
2. **Постоянно включить** - добавляет конфигурацию в APT
