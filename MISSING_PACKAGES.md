# 🔧 Решение проблем с недоступными пакетами в Astra Linux

## Проблема

При запуске скриптов развёртывания вы видите ошибку:
```
E: Для пакета "iotop" не найден кандидат на установку
```

Это означает, что пакет недоступен в репозиториях Astra Linux.

## ✅ Решение

**Скрипты уже исправлены!** Они теперь обрабатывают недоступные пакеты и продолжают работу.

### Что было изменено:

#### scripts/00_prerequisites.sh
- ✅ Перемещены опциональные пакеты (iotop, blktrace, fio) в отдельный цикл
- ✅ Добавлена обработка ошибок - если пакет недоступен, скрипт продолжает работу

#### scripts/01_node_prepare.sh
- ✅ Аналогичные изменения для утилит мониторинга

#### scripts/04_opennebula_install.sh
- ✅ Разделены обязательные и опциональные зависимости
- ✅ Обработка ошибок при установке отдельных пакетов

## 📋 Список известных проблемных пакетов в Astra Linux

| Пакет | Статус | Рекомендация |
|-------|--------|--------------|
| **iotop-c** | ✅ Рекомендуется | Современная версия, вероятно доступна |
| iotop | ❌ Может быть недоступен | Оригинальный (устаревший) |
| blktrace | ❌ Может быть недоступен | Трассировка дисков |
| fio | ❌ Может быть недоступен | Бенчмарк производительности |
| libmysql++-dev | ⚠️ Может быть недоступен | Альтернатива: libmysqlclient-dev |

## 🛠️ Если вы хотите установить эти пакеты вручную

### iotop (мониторинг I/O)
```bash
# ✅ РЕКОМЕНДУЕТСЯ - установите iotop-c (современная версия)
sudo apt-get install -y iotop-c

# Или оригинальный iotop (если iotop-c не доступен)
sudo apt-get install -y iotop

# Альтернатива - используйте iostat
sudo apt-get install -y sysstat
iostat -x 1

# Или используйте dstat
dstat -disk
```

### blktrace (трассировка дисковых операций)
```bash
# Может работать из репозитория Debian/Testing
sudo apt-get install -y blktrace
```

### fio (тест производительности дисков)
```bash
# Вместо этого используйте dd или bonnie++
sudo apt-get install -y bonnie++

# Или установите fio из Debian
sudo apt-get install -y fio
```

## 📊 Мониторинг без отсутствующих пакетов

Если iotop/blktrace/fio недоступны, используйте встроенные команды:

```bash
# Вместо iotop:
iostat -x 1

# Вместо blktrace:
dstat -disk

# Вместо fio:
dd if=/dev/zero of=/tmp/test.img bs=1M count=1000
```
